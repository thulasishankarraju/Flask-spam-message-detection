# Flask Spam Message Detection

This project is a Spam Message Detection Web Application developed using Python, Flask, and Machine Learning.

The system predicts whether a message is Spam or Not Spam by analyzing the text entered by the user.

## Features
- User-friendly web interface
- Spam message prediction
- Machine Learning based text classification multinomialNB
- Flask backend integration
- Fast and simple response system

## Technologies Used
- Python
- Flask
- HTML
- Scikit-learn
- Pandas
- NumPy

## Project Structure
Flask-Spam-Message-Detection/
│
├── spam.py
├── spam.html
├── spam.csv

## How to Run
1. Install libraries like pandas,flask,scikit-learn
2. Run the Flask server
3. Open browser and access localhost
